# 🔐 Cybersecurity Tools & Resources

## 🔎 Recon & OSINT
- **Whois** – Search domain names or IP addresses to view ownership, registration, and DNS details.  
  https://www.whois.com/

- **Discord Server Widget API** – Retrieve public Discord server info using server ID.  
  https://discordapp.com/api/servers/<serverID>/widget.json

---

## 🔓 Encoding / Decoding / Cryptography
- **CyberChef** – All-in-one tool for encoding, decoding, and data transformation.  
  https://gchq.github.io/CyberChef/

- **CacheSleuth Multi Decoder** – Decode Base64, URL, HTML, and more.  
  https://www.cachesleuth.com/multidecoder/

- **dCode ASCII Tool** – ASCII conversions and cipher analysis.  
  https://www.dcode.fr/ascii-code

- **RapidTables Converter** – Convert ASCII, binary, hex, and decimal.  
  https://www.rapidtables.com/convert/number/ascii-hex-bin-dec-converter.html

- **IBM ASCII Conversion Table** – Reference for ASCII formats.  
  https://www.ibm.com/docs/en/aix/7.2?topic=adapters-ascii-decimal-hexadecimal-octal-binary-conversion-table

- **Rumkin Caesar Cipher Tool** – Caesar cipher encoder/decoder.  
  https://rumkin.com/tools/cipher/caesar/

- **Base65536 Decoder** – Decode Base65536 encoded data.  
  https://www.better-converter.com/Encoders-Decoders/Base65536-Decode
